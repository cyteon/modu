## Type Conversion
Modu comes with several functions to convert between types, which can be useful when getting terminal input with `input()`.

[CODE]
// int() is used to convert to ints
print(int("100"));

// float() is used to convert to floats
print(float("1.5"));

// bool is used to convert to bools
print(bool(1));

// str is used to convert to strings
print(str(1));

// type is used to print the type of a value
print(type(1));
