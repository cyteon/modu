## Bitwise Operations
Modu has support for the following bitwise operations:
- `&` (AND)
- `|` (OR)
- `^` (XOR)
- `<<` (left shift)
- `>>` (right shift)
- `~` (NOT)

[CODE]
// bitwise AND operation
print(12 & 10);

// bitwise OR operation
print(12 | 10);

// bitwise XOR operation
print(12 ^ 10);

// bitwise left shift operation
print(12 << 2);

// bitwise right shift operation
print(12 >> 2);

// bitwise NOT operation
print(~12);
